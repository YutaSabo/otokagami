# Product Goal

## プロダクト名

Pronunciation Mirror

日本語名: 発音ミラー

## コンセプト

英文を入れて1回話すだけで、ネイティブには「何の音」に聞こえたかを音素ごとに色で見せ、間違った音だけ日本語の短い助言と静的な図で直し方を示す。

MVPでは、自由入力よりも「提示された毎日の7個を話す」体験を主軸にする。利用者が毎日続けやすく、判定回数と運用コストを抑えながら、通じない音を一点ずつ直すことを狙う。

## 一文での目的

国籍・母語を問わない英語学習者が、毎日7個の短い練習と音素単位のフィードバックで発音を継続的に改善できる iPhone アプリを作る。初期UI・助言は日本語版を先行するが、判定スコアは母語・国籍に依存させない。

## ターゲット

主対象は、日本語を母語とする英語学習者である。

このユーザーは、英語を完璧なネイティブ発音に近づけたいというより、L/R、TH、V/B、母音長、語末子音、子音連結などで「相手に違う単語として聞こえる」「通じない」問題を解決したい。

## 解決する課題

- 自分の発音が、ネイティブにはどの音に聞こえているか分からない。
- 単語全体の点数だけでは、どの音を直せばよいか分からない。
- 英語の発音解説は長く、毎日続かない。
- 日本語話者特有の誤りに合わせた短い助言が少ない。
- 無料ツールは単発利用に向いているが、苦手音の推移や復習まで続かない。

## 提供価値

### 1. 音素単位で見える

Azure 発音評価 API が返す音素スコアと実測音素をもとに、期待IPAと実測IPAを2段で表示する。違う音だけ色で見せることで、ユーザーは「英語っぽいか」ではなく「何の音に聞こえたか」を把握できる。

### 2. 外した音だけ直す

間違った音に対してのみ、1〜2行の日本語助言と静的な口・舌の図を出す。説明は長くしない。間違いゼロの場合は詳細説明を出さず、完了感を優先する。

### 3. 判定は全英語学習者、コンテンツは段階的にローカライズする

MVPの標準判定ロケールは`en-US`とし、`native_language = und`を既定にする。母語は助言ローカライズや問題推薦に将来利用できるが、Azureスコアの補正には使わない。初期UI、助言、パック、直し方ページは日本語版を先行する。

### 4. 毎日7個で続ける

メイン体験は、単語5個と文2個のデイリー練習である。ホームに7個の一覧は出さず、スタート後に1個ずつ提示する。7個終われば「今日は終わり」と分かる構造にする。

### 5. 進捗を貯めて価値にする

判定結果を保存し、音素ごとの習熟度、ストリーク、ヒートマップ、推移グラフ、レベル、バッジ、称号に変換する。これらは追加の従量コストを増やさず、Pro継続の理由になる。

## ポジショニング

Pronunciation Mirror は、総合英会話アプリでも、発音ゲームでも、単語帳でもない。

立ち位置は「狭く、深く、習慣化、低価格」である。会話、カリキュラム、動画、総合学習に広げず、世界の英語学習者が伝わりにくい音を直すことに集中する。

## 差別化

| 観点 | 方針 |
| --- | --- |
| 対象 | 国籍・母語を問わない英語学習者。 |
| 入力 | メインは提示問題、サブとしてPro自由入力。 |

| 助言 | 1音につき短い日本語助言に絞る。 |

## 本当の堀

判定エンジンそのものではなく、母語別にローカライズできる直し方の蓄積と検証である。

MVPでは、頻出の混同ペアに対して静的な直し方ページとテンプレ助言を用意する。未知ケースだけOpenAI APIで短文整形し、結果をキャッシュする。助言の有効性は、再録音スコア改善とユーザー評価で検証する。

## 成功状態

MVPの成功状態は次のとおり。

- ユーザーがホームから1タップでデイリー練習を開始できる。
- 1問ごとに録音、Azure判定、音素別表示、聞き比べ、詳細、直し方まで進める。
- ターゲット音素の習熟度が更新され、翌日以降の出題に反映される。
- ストリーク、ヒートマップ、推移グラフ、レベル/バッジ/称号が判定結果から更新される。
- 8日目以降はPro登録しないと練習できない。
- 自由入力はPro限定であり、デイリー集計を汚さない。
- 音声データはサーバーに保存されない。

## スコープ外

以下は、MVPの目的から外れるため作らない。

- AI会話。
- カリキュラム、レッスン動画、解説動画。
- ネイティブそっくりのアクセントを目指す訴求。
- 発音ゲーム。
- 単語帳。
- `en-US`以外のロケール向けprosody判定。
- 自信度表示。
- 効く順ランキング。
- Android。
- iPad専用UI。
- Webお試しページ。
- 多言語UI。
- UKアクセントの実判定。
- ランキング。
- 成果シェア画像。
- シャドーイング。
